# httpcodes
HTTP状态码含义查询

# Install
```bash
npm install -g @mailzwj/hc
```

# Usage
```bash
用法：hc show <code>

命令：
  hc show <code>  显示状态码详情                                    [aliases: s]

选项：
  --version  显示版本号                                                   [布尔]
  --help     显示帮助信息                                                 [布尔]

示例：
  hc show 200
```
